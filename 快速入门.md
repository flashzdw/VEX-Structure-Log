
# 🚀 VEX 机器人结构改动日志 - 快速入门指南

## 5分钟快速上手

### 第一步：注册账号（1分钟）

1. 打开应用：`http://localhost:5174` 或 `http://1247ug121tl26.vicp.fun`
2. 点击 **"没有账号？立即注册"**
3. 填写邮箱和密码
4. 完成注册！

### 第二步：加入队伍（1分钟）

1. 登录后进入设置页面
2. 点击 **"加入队伍"**
3. 输入队长分享的邀请码
4. 点击 **"加入"**

### 第三步：创建第一条记录（2分钟）

1. 点击首页右上角的 **"新建记录"**
2. 填写信息：
   - 日期：今天
   - 负责人：你的名字
   - 模块：选择你负责的模块
   - 改动原因和内容：简单写写
   - 测试结果：简单描述
3. 点击 **"保存"**

🎉 **完成！你已经会用了！**

## 💡 记录填写小贴士

### 必填项

- ✅ 日期
- ✅ 负责人
- ✅ 模块
- ✅ 改动原因或内容

### 选填但建议填写

- 📷 上传照片（强烈建议！）
- ⭐ 评分
- 🏷️ 里程碑标记（重要版本）
- 📝 测试结果和问题

### 好的记录示例

```
改动原因：
抓手太松，比赛时会掉东西

改动内容：
把弹簧从1.5kg换成2.5kg，增加力度

测试结果：
抓取更稳了！连续10次测试没有掉东西

遇到问题：
力度太大，偶尔会把物体夹变形

下一步计划：
测试2.0kg的弹簧，找到平衡点
```

## 🆘 遇到问题？

### 常见问题快速解决

| 问题 | 解决方法 |
|------|----------|
| 登录失败 | 检查账号密码是否正确，或重新注册 |
| 注册失败 | 确保邮箱格式正确，密码至少8位 |
| 页面打不开 | 检查网络，确认服务正在运行 |
| 保存失败 | 检查必填项是否都填了 |
| 照片上传失败 | 照片太大？建议每张 &lt; 5MB |
